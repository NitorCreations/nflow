## Development

Installation

```
npm install
```

Starting dev server

```
npm start
```

Point your browser to http://localhost:3000/

Use `npm`. `yarn` has caused problems.

### Auto formatting

This repository has code auto formatting enabled. Auto formatter runs before commit.
